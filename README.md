## Soal

Buatlah sebuah program dalam bahasa pemrograman pilihanmu yang menerima input dari pengguna berupa beberapa bilangan bulat. Program tersebut 
harus menghitung dan menampilkan nilai rata-rata dari bilangan-bilangan yang diinputkan, serta menentukan apakah setiap angka berada di atas 
atau di bawah rata-rata. Jumlah bilangan yang diinputkan tidak ditentukan. Input berakhir ketika pengguna memasukkan angka 0.

## Cara Menjalankan

1. Clone Repository
2. Pastikan Node.js sudah terinstall
3. Jalankan perintah berikut di terminal:

```
node rata_rata.js
```

## Contoh Penggunaan

## Input:

```
10
20
30
0
```

### Output:

```
Rata-rata: 20
10 di bawah rata-rata
20 sama dengan rata-rata
30 di atas rata-rata
```
